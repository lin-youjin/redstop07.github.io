《Git-it指令小抄》

小提示

方向上鍵 可重複上一條指令
方向下鍵 可重複下一條指令

初始設定

$ git config --global設定區域變數(?)

git config --global user.name "yourname"設定你的名字
git config --global user.email "youremail"設定你的電郵
git config --global user.username "yourGitHubAccount"連動GitHub帳號

基礎指令

ls list
cd 移動使用者目前位置
cp copy
mv move
rm remove
rmdir remove directories
mkdir create a new directory

連動資料夾以及儲存變動

git init將資料夾加上git功能
git status查看檔案狀態
git add foldername: 準備commit一個檔案
git add .:準備commit所有檔案
git diff: check out the difference
git commit -m "yourmessage": commit(儲存)你的變更並附上簡短說明
git remote add origin <URLFROMGITHUB>: 遠端控制GitHub repository
git remote set-url origin <URLFROMGITHUB>: 效果同上(如果下載了GitHub for Window)
g it push origin master: 把分支master 傳送到新增的remote
git clone: 複製專案
git remote add upstream <想上傳變更的網址>: 將分支暫存在向<想上傳變更的網址>的遠端
git remote -v: 檢查導向網址以及輸出網址
git branch: 新增分支(後面直接加分支名字)
git checkout: 切換分支到(直接打分支名)
—紀錄修改小撇步—
git status
git add
git commit -m “”
git push origin

git pull 更新其他的使用者對分支進行的更改

安裝Hexo

在你的電腦打開git.bash或是命令提示字元視窗(cmd)

輸入下載hexo的指令碼

小註:

  • npm: node package manerger，以JavaScript編寫，為Node.js預設的軟體套件管理系統，使用者可以上傳自己編寫的套件軟體，提供不同功能的套組下載。
  • cli: command line interface命令行介面，意即使用非圖標式的鍵盤輸入命令方式。
